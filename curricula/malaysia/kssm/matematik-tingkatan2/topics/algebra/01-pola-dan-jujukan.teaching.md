# Pola dan Jujukan (Patterns and Sequences) — Teaching Notes

## Overview
This topic introduces students to the transition from visual or numeric pattern spotting into algebraic generalization. In KSSM Form 2, this chapter sets up later work on formulae and functions by helping students move from recursive thinking to explicit rules.

## RPT/DSKP Anchors
- Chapter: Bab 1 (Static RPT source, Week 2-3)
- SK: 1.1, 1.2, 1.3
- SP focus: 1.1.1, 1.2.1, 1.2.2, 1.3.1, 1.3.2, 1.3.3

## Prerequisites Check
- Students can represent unknowns with variables (F1-05).
- Students are comfortable with basic integer operations and simple algebraic expressions.

## Teaching Sequence

### 1. Pattern Recognition in Context (15 min)
Start with concrete contexts: tile patterns, seating rows, or stacked matchsticks. Ask students to describe how each stage changes.

### 2. Sequence Vocabulary and Structure (20 min)
Define terms: term, position, common difference, rule. Contrast finite patterns versus extendable sequences.

### 3. Recursive to Explicit Rules (25 min)
Guide learners to move from "add 3 each time" to an expression for term-n, then test across several n values.

### 4. Generalization and Justification (20 min)
Ask students to explain why a formula works, not only compute answers. Encourage using words + symbolic form.

### 5. Problem Solving (20 min)
Use mixed tasks (tables, diagrams, word problems) where students choose strategy and justify the sequence rule.

## Common Misconceptions
| Misconception | Why It Happens | Remediation |
|---|---|---|
| "Pattern" means only visual shape patterns | Prior primary exposure is mostly visual | Use both visual and numeric pattern sets from lesson start |
| Uses recursive rule when question asks nth term | Students focus on local change only | Require answers in both recursive and nth-term forms |
| Substitutes wrong n position (starts at 0 or 2) | Indexing confusion | Force a term-position table before formula substitution |

## Engagement Hooks
- "Mystery Rule" game: groups infer hidden sequence rules from 4-5 shown terms.
- Daily-life pattern sprint: identify sequence patterns in school timetable, stairs, or canteen pricing bundles.

## Assessment Guidance
- Mix "continue the sequence" and "find nth term" prompts.
- Include at least one explain-your-reasoning item.
- Use error-analysis tasks where students fix an incorrect generalization.
