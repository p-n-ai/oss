# Ketaksamaan Linear (Linear Inequalities) - Teaching Notes

## Overview
Chapter F1-07 extends equation reasoning to inequalities and solution sets. Students learn to express constraints, solve them correctly, and interpret solution intervals.

## DSKP Coverage Snapshot
- 7.1 Inequalities
- 7.2 Linear inequalities in one variable

## Prerequisites Check
- F1-05 algebraic expressions
- Recommended: F1-06 linear equations

## Teaching Sequence

### 1. Meaning of inequality symbols (15 min)
- Compare numeric values and state inequality relations.
- Move from number comparison to algebraic inequality.

### 2. Inequality properties (20 min)
- Converse/transitive ideas and inverse relationships.
- Arithmetic operation effects on inequality direction.

### 3. Forming inequalities from context (20 min)
- Convert daily-life constraints into one-variable inequalities.
- Translate back from symbolic form to plain language.

### 4. Solving and representing solution sets (25 min)
- Solve one-variable inequalities step by step.
- Represent solution sets on number lines.

### 5. Simultaneous inequalities (20 min)
- Solve two constraints in one variable.
- Identify the intersection region as the final solution set.

## Common Misconceptions
| Misconception | Why it happens | Remediation |
|---|---|---|
| Forget to reverse inequality when multiplying/dividing by negative | Overgeneralized equation rules | Drill explicit negative-operation checkpoints |
| Confuses `>` and `<` direction | Symbol orientation issue | Use arrow/number-line visual anchors |
| Wrong open/closed endpoint on number line | Symbol-endpoint mismatch | Pair examples (`<`,`>`) vs (`<=`,`>=`) repeatedly |
