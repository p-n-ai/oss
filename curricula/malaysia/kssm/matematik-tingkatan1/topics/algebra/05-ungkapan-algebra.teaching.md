# Ungkapan Algebra (Algebraic Expressions) - Teaching Notes

## Overview
Chapter F1-05 introduces variables and algebraic expressions, then extends to operations on expressions. Students move from arithmetic-only statements to symbolic reasoning used throughout later algebra topics.

## DSKP Coverage Snapshot
- 5.1 Variables and algebraic expressions
- 5.2 Algebraic expressions involving basic arithmetic operations

## Prerequisites Check
- Basic arithmetic operations (+, -, x, /)
- Understanding unknown values in primary-school word problems

## Teaching Sequence

### 1. Variables as unknowns (15 min)
- Use short story contexts and replace unknown quantity with letters.
- Emphasize that variable values can be fixed or varying depending on context.

### 2. Structure of expressions (20 min)
- Terms, coefficients, constants, like and unlike terms.
- Classify sample expressions before performing operations.

### 3. From words to algebra (20 min)
- Convert daily situations into algebraic expressions.
- Practice phrase mapping (for example: "3 more than x" -> x + 3).

### 4. Operations with expressions (25 min)
- Add/subtract by collecting like terms.
- Extend to repeated multiplication and one-term multiply/divide.

### 5. Substitution and interpretation (15 min)
- Evaluate expressions when variable values are given.
- Connect numeric result back to the original context.

## Common Misconceptions
| Misconception | Why it happens | Remediation |
|---|---|---|
| `3x + 2y = 5xy` | Students combine unlike terms | Use color-coding/algebra tiles to separate term types |
| `2x` read as a two-digit number | Symbol reading confusion | Write `2 x x` first, then compress to `2x` |
| Sign errors during substitution | Weak bracket discipline | Force substitution with brackets, then simplify |

## Engagement Hooks
- Mobile plan pricing model: base fee + per-use variable cost
- Canteen combo pricing with one unknown item
- Mystery-number challenge in pairs

## Assessment Guidance
- Start with substitution, then simplify, then word-to-expression translation
- Include one "spot the error" item targeting unlike-term addition
