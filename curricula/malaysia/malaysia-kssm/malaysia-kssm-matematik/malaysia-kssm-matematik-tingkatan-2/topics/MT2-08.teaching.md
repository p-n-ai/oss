# Graf Fungsi — Teaching Notes

## Overview
In this topic, students will delve into understanding and interpreting functions. They'll learn how to identify functions through various representations such as ordered pairs, tables, graphs, and equations. They'll also practice constructing value tables for linear and nonlinear functions and drawing graphs using given scales. Finally, students will interpret function graphs and solve problems involving these graphs.

## Prerequisites Check
Students should be familiar with basic algebraic concepts, understanding coordinates, and simple graph plotting. Familiarity with linear equations and basic arithmetic operations is also essential.

## Teaching Sequence & Strategy

### 1. Introduction to Functions (15 min)
Introduce the concept of a function as a special kind of relation between two sets where each input is related to exactly one output.
- **Strategies:** Use visual aids and diagrams to illustrate input-output relationships.
- **Check for Understanding (CFU):** Ask students to describe a function in their own words.
- **The Trap:** Misunderstanding that a function can have the same output for different inputs.

### 2. Identifying Functions (20 min)
Guide students through identifying functions using ordered pairs, tables, and graphs.
- **Strategies:** Use examples and non-examples; employ a T-chart for ordered pairs.
- **Check for Understanding (CFU):** Present a set of ordered pairs and ask if it represents a function.
- **The Trap:** Believing that every relation is a function.

### 3. Constructing and Drawing Graphs (25 min)
Teach students how to construct value tables and plot linear and nonlinear graphs.
- **Strategies:** Hands-on graph plotting using graph paper; use a step-by-step approach.
- **Check for Understanding (CFU):** Have students plot a simple function and explain their steps.
- **The Trap:** Confusing the x and y axes when plotting points.

### 4. Interpreting Graphs (20 min)
Help students analyze and interpret different aspects of function graphs.
- **Strategies:** Discuss real-world scenarios; use graph interpretation questions.
- **Check for Understanding (CFU):** Ask students to describe what a graph tells us about the function.
- **The Trap:** Not understanding the significance of slope and intercepts.

### 5. Solving Problems with Graphs (20 min)
Engage students in problem-solving activities using function graphs.
- **Strategies:** Use word problems; encourage group discussions.
- **Check for Understanding (CFU):** Solve a problem together and then individually.
- **The Trap:** Overlooking graph details that are essential for solving the problem.

## High Alert Misconceptions
| Misconception | Why Students Think This | How to Fix |
|---------------|-------------------------|------------|
| All graphs are functions | They assume any graph is a function | Use vertical line test to explain |
| Functions can have multiple outputs for one input | Misinterpretation of output definition | Reinforce one input-one output rule |

## Engagement Hooks
- Use local public transportation schedules to illustrate functions as they relate departure times (input) to arrival times (output).

## Assessment Guidance
When assessing understanding, check for correct identification of functions, accurate graph plotting, and logical problem-solving. Encourage students to explain their thinking process.
